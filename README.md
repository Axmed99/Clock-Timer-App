# Clock & Timer App ⏰

A modern and stylish web-based **Clock & Timer App** built using **HTML, CSS, Bootstrap, and JavaScript**. This app includes multiple features such as a **Clock, Timer, Alarm, and Stopwatch**, making it a perfect time management tool.

## Features
- 🕒 **Clock** – Displays real-time digital clock with date.
- ⏳ **Timer** – Set a countdown timer with notifications.
- 🔔 **Alarm** – Configure alarms with sound alerts.
- ⏱️ **Stopwatch** – Track time efficiently with start/stop/reset.

## Technologies Used
- **HTML5** – Structure of the application.
- **CSS3 & Bootstrap** – Styling and responsiveness.
- **JavaScript** – Functionality and dynamic updates.
